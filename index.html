This page demonstrates dynamic form generation powered by Corticon.js decision services. The forms are created dynamically based on rules defined in the decision service, acting as the model for a generic form component.

Features include:

  • Conditional questions based on user inputs and external data.
  • Support for rendering forms using either simple HTML controls or the KendoUI component library.
  • Real-time validation and dynamic updates based on decision service responses.

Use the dropdown menu above to select a sample form, and navigate through the form using the "Start," "Next," and "Previous" buttons. The trace panel below shows the input, output, and accrued form data for each decision service call.
